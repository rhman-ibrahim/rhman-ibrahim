## Featured Projects

### Brochure UI

#### Anchor

Tech: React, Vike, TailwindCSSAnchor transforms traditional Business Model Canvas (BMC) into an interactive, multilingual landing page. It translates each BMC section into dynamic React components, offering a fully responsive and engaging experience.

### Identity API

#### Authen

Tech: Django, DRF, JWTA powerful authentication API with custom user modeling, JWT-based authentication, and activity tracking. Designed for secure and scalable identity management.

#### Accord

Tech: Django, Django Channels, WebSocketsA real-time messaging API that enables instant, reliable communication using WebSockets and Django Channels.

### Layers UI

#### Schema

Tech: React, Vite, and CSS fmanaging components, modals, panels, drawers, forms, and notifications. Includes customizable themes and a structured approach to UI development.

### Semantic AI

#### Sottile

Tech: TensorFlow, React, DjangoAn AI-powered violin practice assistant that analyzes audio recordings to provide intonation, rhythm, and expression feedback. Uses machine learning to help musicians improve their technique efficiently.

#### Query

Tech: Python, Django, SentenceTransformers, FAISSA standalone API for Markdown-based documentation Q&A. Uses semantic search with embeddings to provide relevant, conversational responses based on documentation files. Designed for fast and accurate information retrieval.
